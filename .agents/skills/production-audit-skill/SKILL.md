# Production Audit Skill
- **Review**: Conduct architecture review for boundary violations.
- **Safety**: Perform a panic audit (lack of `unwrap`/`expect`).
- **Dependencies**: Audit crate count and licenses.
- **Runtime**: Verify startup logs and healthy execution.
- **Verification**: Complete the final release checklist.
